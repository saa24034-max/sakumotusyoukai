# マイ作物紹介サイト

私が家庭菜園や学校の授業で挑戦してみたい大好きな2つの作物（さつまいも・とうもろこし）の魅力と栽培のポイントをまとめた紹介ウェブサイトです。

## 💡 このサイトの工夫した点（3つのアピールポイント）

### 1. 視認性と作物への愛着を高めるカラーデザイン（CSS装飾）
サイト全体のテーマカラーとして、紹介する作物をイメージした色彩設計（さつまいもの「鮮やかな紫」と、とうもろこしの「みずみずしい黄」）を取り入れました。背景には目に優しい薄黄色を敷き、コンテンツを白いカード型のボックスにまとめることで、すっきりと読みやすいモダンなデザインに仕上げています。

### 2. スマホ対応（レスポンシブ）と快適なナビゲーション
スマートフォンとパソコンのどちらからアクセスしてもレイアウトが崩れないよう、メディアクエリ（`@media`）を使用したレスポンシブ対応を行いました。特に画面下部のボタンエリアは、スマホ画面では縦並び、PC画面では横並びに自動で切り替わります。また、ページ上部に固定されるヘッダーナビゲーションを設置し、目的の作物情報へ瞬時にスクロールできるページ内リンクを実装しました。

### 3. JavaScriptを用いたインタラクティブな動的仕掛け
ユーザーがサイトをより楽しめるよう、最下部に「栽培への意気込みを表明する！」というJavaScript連動型のインタラクティブなボタンを設置しました。ボタンをクリックすると、応援のアラートメッセージが表示されるだけでなく、ボタン自体の文字が「🔥 豊作を目指して栽培スタート！」へと変化し、背景色も引き締まった黒へと動的に切り替わることで、視覚的な楽しさをプラスしています。
